# Лабораторная работа №7: Изучение Flexbox

## Цели работы
- Ознакомление с концепциями и свойствами Flexbox для создания гибких и адаптивных макетов.
- Изучение методов центрирования элементов с помощью Flexbox.
- Практическое применение Flexbox для построения навигационных меню, карточек товаров и центрированных блоков.

## Что изучалось
### 1 Flexbox
- Свойство `display: flex;` и его роль в создании гибких контейнеров.
- Свойство `flex-wrap: wrap;` для переноса элементов на новую строку.
- Свойство `gap` для задания промежутков между элементами.
- Свойство `flex` для управления размерами элементов.
- Свойство `align-self` для выравнивания отдельных элементов по вертикали.
- Свойство `justify-content` для распределения элементов по горизонтали.
- Центрирование элементов с помощью `justify-content: center;` и `align-items: center;`.

### 2 Примеры использования Flexbox
- Создание навигационного меню, в котором элементы равномерно распределены по горизонтали.
- Размещение карточек товаров в виде колонок, которые автоматически переносятся при необходимости.
- Центрирование блока внутри контейнера по горизонтали и вертикали.

## Практические примеры из работы
- **Навигационное меню**: горизонтальное расположение ссылок с равномерным распределением.
- **Карточки товаров**: вертикальное расположение карточек с переносом на новую строку при нехватке места.
- **Центрирование**: размещение блока по центру страницы или контейнера.

## Итог
Работа позволила понять основные принципы построения адаптивных макетов с помощью Flexbox и CSS, а также научиться быстро создавать структурированные и центрированные элементы
